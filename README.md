## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## After starting the development server needs to import movies
go to this link to import some dummy movies
[Import Movies](http://localhost:3006/api/v1/movie/import)

### Note
Don't forget to update these environment variables. Use your own values
```sh
MONGODB_URI=mongodb://127.0.0.1:27017/nestjs
JWT_SECRET=helloworld
JWT_EXPIRES_IN=7 # in day
```

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
